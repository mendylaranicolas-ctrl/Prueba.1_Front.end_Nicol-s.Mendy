## 🎯 Descripción del Proyecto
Este proyecto es una aplicación web de una sola página (**Single Page Application**) diseñada como portafolio profesional. El objetivo principal es demostrar habilidades sólidas en maquetación semántica, diseño responsivo y control de versiones, aplicados a mi perfil como futuro **Ingeniero en Ciberseguridad**.

> "La seguridad y la tecnología no tratan solo de máquinas, sino de resolver problemas de forma creativa."

---

## 🛠️ Stack Tecnológico
* **Maquetación:** HTML5 con etiquetas semánticas para SEO y accesibilidad.
* **Estilos:** CSS3 avanzado (Flexbox, CSS Grid).
* **Tipografía:** Google Fonts (Sekuya & System UI).
* **Herramientas:** Git para el control de versiones y GitHub para el despliegue.

## ✨ Características Principales
-   **Diseño Responsivo:** Adaptabilidad total en móviles, tablets y desktops mediante Media Queries.
-   **Navegación Inteligente:** Menú superior fijo (Sticky Navbar) con desplazamiento suave (Smooth Scroll).
-   **UX/UI Moderna:** Efectos de escala tipo Apple en la galería de proyectos y barras de progreso dinámicas en la sección de habilidades.
-   **Validación de Datos:** Formulario de contacto integrado con validaciones nativas de HTML5.
-   **Optimización Visual:** Scrollbar personalizado y paleta de colores coherente (Azul Marino y Naranja Corporativo).
